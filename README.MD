PROGRAMA

FUNCIONAMIENTO DEL PROGRAMA
Este programa ejecuta de forma automatica la cotizacion del seguro. En el cual el mismo solicitante provee la información
y obtiene por si mismo el resultado que necesita.

PARTES QUE SE CONCIDERAN
Este programa concidera de los factores disponibles para la ejecución, al analizar las respuestas del solicitante.

Por ejemplo, si el solicitante declara estar casado el programa conciderará esta respuesta y analizará un nuevo dato para 
la cotización.

Las variables dispuestas en este programa actuaran como retenciones sobre el precio base del seguro, en los casos de edad del
solicitante y del conyuge si estuviera casado/a, poseyendo diversos rangos entre las posibles edades que pudieran ser ingresadas;
concidera tambien la cantidad de hijos con una retención sobre cada uno de ellos, si los tuviera.

Añadiendo los resultados de cada retención y el precio base del seguro e imprimiendo el total para que el solicitante pueda verla.

QUE SE PODRIA MEJORAR
En este punto de la clase podriamos conciderar que el aspecto ideal a mejorar podria ser el diseño de la página, ya que asi 
podría ser mas atractiva al usuario. Tambien podriamos mejorar el hecho de que el programa no solo sirviera para 
cotizar el precio de un seguro, sino ampliar sus funciones a por ejemplo contratar y pagar el seguro.
